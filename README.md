♥♥Introduction♥♥

SoulMate AI is a fully offline, privacy-first AI companion that runs on your own computer. It uses Ollama to run a local LLM (qwen2:1.5b) and stores all data locally. No API keys, no cloud, no tracking.



♥♥Requirements♥♥

\- \[Ollama](https://ollama.com) installed

\- Model: `qwen2:1.5b` (download with `ollama run qwen2:1.5b`)



♥♥How to Run♥♥

1\. Clone this repository or download `Main.java`

2\. Open `Main.java` in Eclipse (or any Java IDE with JDK 11+)

3\. Run as Java Application

4\. Follow the setup prompts (choose language, partner name, personality, etc.)

5\. Start chatting!



> All configuration and memory files (`config.txt`, `memory.txt`) are auto-generated at first run.



♥♥Commands♥♥

\- `exit` - quit the program

\- `restart` - reset all settings and clear memory (auto-restart)


♥♥Notes♥♥

\- The AI responds in the same language you use (Chinese or English)

\- Only `Main.java` is source code; everything else is generated at runtime

# SoulMate AI

♥♥简介♥♥
灵魂伴侣AI是一个完全本地运行、保护隐私的 AI 伴侣。它使用 Ollama 在本地运行大语言模型（qwen2:1.5b），所有数据都保存在你自己的电脑上。无需 API Key，无需联网，无追踪。



♥♥运行要求♥♥

\- 已安装 \[Ollama](https://ollama.com)

\- 下载模型：`ollama run qwen2:1.5b`



♥♥如何运行♥♥

1\. 克隆本仓库或下载 `Main.java`

2\. 用 Eclipse（或任何支持 JDK 11+ 的 Java IDE）打开 `Main.java`

3\. 以 Java 应用程序方式运行

4\. 按提示完成首次设置（选择语言、伴侣名字、性格等）

5\. 开始聊天！



> 所有配置和记忆文件（`config.txt`、`memory.txt`）在首次运行时自动生成。


♥♥指令♥♥

\- `exit` - 退出程序

\- `restart` - 重置所有配置并清空记忆（自动重启）



♥♥ 注意事项♥♥

\- AI 会根据你输入的语言回复（中文或英文）

\- 源码只有 `Main.java`，其他文件均为运行时自动生成
